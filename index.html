PHANTOM RECALL

Technical Analysis

Identified a critical flaw in LLM reasoning where models revert to training data memorization when faced with "familiar but altered" logic puzzles.

  • Benchmark Development: Created a dataset of 500+ perturbations of classic logic problems (Monty Hall, Bridge and Torch).
  • Recall Bias: Proven that LLMs often ignore explicit logic changes in prompts if they resemble high-frequency training examples.
  • Taxonomy: Developed an error-mapping system to categorize "Reasoning Shortcuts" in Transformer models.

HISTORICAL-AWARE PERCEPTION

Technical Analysis

Addressing the failure of autonomous vehicle perception in blizzards by utilizing temporal cross-attention between current frames and past clear traversals.

  • Architecture: Replaced baseline ResNet-18 encoders with Swin-Transformers to capture global context in heavy fog.
  • Temporal Cross-Attention: Implementation of a module that retrieves "Historical Frames" via GPS metadata to identify road geometry hidden by snow.
  • Pre-training: Utilizing contrastive learning on synthetic adverse weather data generated from the Ithaca365 dataset.

MICROCAML COMPILER

  • AST Engineering: Developed an Abstract Syntax Tree (AST) framework to handle recursive functions and closure bindings.
  • Evaluation: Implemented a big-step semantics interpreter to execute complex functional programs.
