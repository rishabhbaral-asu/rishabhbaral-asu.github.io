Phantom Recall

Benchmarking memorization vs. reasoning

An investigation into whether LLMs reason through logic puzzles or perform high-dimensional lookup of training patterns.

  • Methodology: developed a counterfactual logic dataset where standard puzzles were permuted to invalidate known training patterns.
  • Metrics: evaluated accuracy \(A\) as a function of n-gram overlap \(O\), identifying a sharp drop-off when memory fails.
  • Key finding: identified the Phantom Limit, where model confidence \(P(y|x)\) remains high despite logical contradictions.

Moneyball LLM

Tabular summarization in long-context narratives

Sports narratives are dense with tabular data hidden in prose. This project implements a retrieval pipeline to extract and summarize box-score statistics from long-form play-by-play text.

  • Architecture: hybrid long-context windowing with a cross-encoder re-ranker for statistically significant game events.
  • Extraction: transformed unstructured narratives into JSON and DataFrames using constrained decoding and Pydantic validation.
  • Application: reduced 10,000+ word game logs into narrative summaries with 94% statistical accuracy.

Edge-Optimized VLM

Diagnosing geographic domain shift in autonomous driving

An end-to-end, edge-deployable vision-language model of roughly 50M parameters designed to diagnose geographic covariate shift in autonomous driving. The system classifies weather, time of day, and scene topology using BDD100K and curated out-of-distribution diagnostics.

  • Architecture: paired a ConvNeXt-Tiny backbone with a 49-token spatial projection layer to preserve \(7 \times 7\) geometric priors for an SDPA Transformer decoder.
  • Training strategy: used homoscedastic uncertainty weighting to balance multi-task gradients across lower- and higher-entropy targets.
  • Optimization: used differential learning rates, \(1 \times 10^{-5}\) for the backbone and \(1 \times 10^{-4}\) for the decoder, with AdamW.
  • Hardware: trained the pipeline on Apple Silicon M3 with Metal Performance Shaders acceleration.
  • Analysis: reached 68.23% semantic exact match on 10,000 ID images and used cross-attention analysis to inspect failure modes.
